About Memory Management Visualization
Memory Management Techniques

Paging: A memory management scheme that eliminates the need for contiguous allocation of physical memory. In this scheme, the operating system retrieves data from secondary storage in same-sized blocks called pages.

Segmentation: A memory management technique that divides programs into segments of different sizes. Each segment is a logical grouping of the program's code or data.

Virtual Memory: A memory management capability that uses hardware and software to allow a computer to compensate for physical memory shortages by temporarily transferring data from random access memory (RAM) to disk storage.

Page Replacement Algorithms

FIFO (First-In-First-Out): The oldest page in memory is the first one to be replaced.

LRU (Least Recently Used): The page that has not been accessed for the longest period of time is replaced.

Key Metrics
  • Page Fault: Occurs when a program accesses a page that is not currently in the main memory, requiring the operating system to fetch it from disk.
  • Hit Ratio: The percentage of memory accesses that found the desired page in memory.
  • Memory Utilization: The percentage of total memory that is currently allocated.
